Do It Yourself: A Multipart Assignment

Christine Julien, The University of Texas at Austin

 

Let's put together our new found knowledge and see if we can write an Android app to toggle (turn on and off) an LED on a breadboard. To accomplish this application, you will need:

  • A partner
  • An Android phone
  • A computer with Android Studio
  • A raspberry pi
  • A computer with VNC to the RPi
  • A GPIO breakout kit
  • A breadboard
  • An LED
  • A 390 ohm resistor (please use the resistor correctly, otherwise, we risk the RPi)

You should create an Android app that has two buttons, one labeled "on" and one labeled "off". These should be used to turn an LED on the breadboard ON (and OFF) using a bluetooth connection between the Android app and the RPi.

This circuit diagram is the only major hint I'm giving, just because we haven't learned a ton about circuits and resistors and their purpose, so it would be difficult to get right. Of course, you're using the breakout board, so you'll need to map those pins onto those on your breakout board.

The other minor hint is to start the Android program from the one we wrote in the communication module. But you need the buttons in the Android app to toggle, too. That is, you want the ON button to be enabled only when the LED is off and you want it to be disabled (not clickable) when the LED is on. The reverse should be true for the OFF button.

[Optional] Have some extra time?

Here's another step if you have some extra time. Build the temperature or motion or light sensor back up on the breadboard (without affecting the LED at all). Add a button to your Android application to retrieve the temperature value. The LED on/off buttons should continue to work exactly the same as above.
